setup

Module to represent a mixture of fluids, it’s used to save the multiple component’s properties to be used by different subroutines. It also includes a set of subroutines to read the data and save it in the module.

System of units: This units are asumed, if the user wants to use another system, the RGAS constant should be changed at the module constants

  • Volume: Liter
  • Pressure: bar
  • Temperature: Kelvin

Uses

    • constants
